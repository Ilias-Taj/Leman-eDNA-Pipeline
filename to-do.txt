KEY OBSERVATIONS
  - PR2 is clearly the best DB for 18S and JEDI markers
  - COI classification is weak across all DBs -- likely because
    general-purpose COI DBs lack local freshwater/soil taxa coverage
  - Soil COI has 31K OTUs but almost none resolve to genus
  - Isospora (coccidian parasite) dominates 18S and JEDI -- real signal
    or primer artifact? Needs verification.
  - "Homo" in soil COI top genera = contamination (skin/breath DNA)

1. COMPILE LOCAL SWISS TAXA REFERENCE LIBRARY
   Priority: HIGH -- the biggest gap is COI genus-level resolution
   Steps:
     a) Define target species list
        - Download Swiss freshwater + soil eukaryote checklists from:
          * info fauna (CSCF) — Swiss center for fauna cartography
          * SwissFungi / SwissLichens databases
          * GBIF Switzerland occurrence data (filter: Geneva/Vaud/Leman)
          * FOEN Red Lists (fish, invertebrates, plants)
        - Include: fish, macroinvertebrates, diatoms, zooplankton,
          phytoplankton, fungi, soil protists relevant to Leman basin

     b) Extract reference sequences from NCBI/BOLD
        - Use rentrez (R) or Biopython Entrez to pull COI/18S/ITS for
          each species on the Swiss list
        - Pull from BOLD Systems API for COI barcodes (barcode gap-checked)
        - Filter: >400bp for COI, >500bp for 18S, remove low-quality
        - Deduplicate with cd-hit (99% identity)

     c) Format for SINTAX
        - Convert to SINTAX-compatible FASTA with taxonomy headers:
          >seqID;tax=d:Eukaryota,p:Arthropoda,c:Insecta,...,s:Species_name
        - Ensure 7-rank lineage (Domain to Species) is complete
        - Validate with vsearch --sintax_summary

     d) Merge with existing DBs
        - Combine Swiss library with PR2 (for 18S) and Porter/eKOI (for COI)
        - Remove duplicate accessions, keep Swiss entries for conflicts
        - Build .udb index with: vsearch --makeudb_usearch swiss_coi.fasta --output swiss_coi.udb

     e) Validate
        - Run pipeline on existing data with new Swiss DB
        - Compare genus-level resolution vs current DBs
        - Check: does Soil COI resolution improve?
        - Cross-validate: do known Leman species appear?

     Tools needed: Biopython, rentrez (R), BOLD API, cd-hit, vsearch
